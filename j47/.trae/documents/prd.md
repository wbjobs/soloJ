## 1. 产品概述

网页版点云查看器，支持用户上传 .ply/.obj 格式的点云文件，通过 WebGL 进行高性能渲染，并提供3D框选工具用于区域选择与高亮。面向3D开发者、研究人员和数据分析人员，解决点云文件在线可视化与交互分析的需求。

## 2. 核心功能

### 2.1 用户角色

无需角色区分，所有用户拥有完整功能访问权限。

### 2.2 功能模块

1. **查看器主页面**：文件上传、3D点云渲染、视角控制、框选工具、信息面板

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 查看器主页面 | 文件上传区 | 支持拖拽或点击上传 .ply / .obj 文件，显示上传进度与文件信息 |
| 查看器主页面 | 3D渲染视口 | 使用 Three.js WebGL 渲染点云，支持鼠标左键旋转、滚轮缩放、右键平移 |
| 查看器主页面 | 框选工具 | 在3D视图中拖拽出立方体区域，高亮显示区域内选中的点，显示选中点数量 |
| 查看器主页面 | 工具栏 | 包含：框选模式切换、重置视角、点大小调节、颜色模式切换 |
| 查看器主页面 | 信息面板 | 显示点云统计信息（总点数、边界范围、选中点数） |

## 3. 核心流程

用户打开页面 → 上传点云文件(.ply/.obj) → 后端接收并解析为JSON/Buffer → 前端接收数据并创建Three.js点云对象 → WebGL渲染显示 → 用户交互（旋转/缩放/平移） → 用户切换框选模式 → 在3D视图中拖拽定义立方体区域 → 区域内点高亮显示 → 显示选中点统计信息


## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深色系（#0a0e17 暗夜蓝黑）+ 青绿强调色（#00ffa3 霓虹绿）
- **辅助色**：#1a1f2e（面板背景）、#2a3040（卡片背景）、#ff6b35（选中高亮橙）
- **按钮风格**：圆角微光按钮，hover时发光效果
- **字体**：JetBrains Mono（数据展示）+ Space Grotesk（UI文字）
- **布局风格**：左侧工具栏 + 中央3D视口 + 右侧信息面板
- **图标风格**：线性图标（Lucide），与深色主题统一

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 查看器主页面 | 文件上传区 | 拖拽区域，虚线边框，居中图标+文字，拖入时边框高亮动画 |
| 查看器主页面 | 3D渲染视口 | 全屏3D画布，深色背景，底部渐变遮罩，顶部状态指示 |
| 查看器主页面 | 工具栏 | 左侧竖排工具按钮，图标+tooltip，选中状态发光 |
| 查看器主页面 | 框选立方体 | 半透明青绿色立方体线框，选中点橙色高亮 |
| 查看器主页面 | 信息面板 | 右侧半透明面板，数据以等宽字体展示，数值动画过渡 |

### 4.3 响应式设计

桌面优先设计，3D视口占满可用空间。工具栏和信息面板可折叠以适应较小屏幕。触控设备支持手势旋转缩放。

### 4.4 3D场景设计

- **环境**：深色背景，网格地面参考线（微弱透明度）
- **光照**：环境光 + 点光源，点云使用顶点着色
- **相机**：透视相机，初始位置自适应点云包围盒
- **点渲染**：圆形粒子纹理，默认白色，选中橙色，可调点大小(1-10px)
- **交互动画**：框选立方体出现时淡入动画，选中点闪烁高亮
- **后处理**：微弱泛光效果(Bloom)增强视觉质感
