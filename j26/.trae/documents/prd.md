# WASM Python 运行环境与可视化调试器 - 产品需求文档

## 1. 项目概述

### 1.1 项目背景
创建一个基于WebAssembly的Python运行环境，允许用户在浏览器中直接编写、执行和调试Python代码，无需本地安装Python环境。该项目结合了现代Web技术和Python生态系统，为开发者提供便捷的在线开发体验。

### 1.2 项目目标
- 在浏览器中实现完整的Python代码执行环境
- 提供专业级的可视化调试工具
- 支持Python标准库核心功能
- 通过WASI实现虚拟文件系统访问
- 提供预编译的常用Python包（如numpy）

### 1.3 目标用户
- Python学习者和教育工作者
- 需要快速原型验证的开发者
- 希望在浏览器中演示Python代码的技术人员
- 在线编程教育平台的用户

---

## 2. 功能需求

### 2.1 核心功能模块

#### 2.1.1 代码编辑器
- **Monaco Editor集成**：提供VS Code级别的编辑体验
- **Python语法高亮**：完整的Python语法着色
- **行号显示**：可点击的行号用于设置断点
- **代码自动补全**：基础的Python代码补全功能
- **主题支持**：支持深色/浅色主题切换

#### 2.1.2 Python运行时
- **WASM执行**：基于Pyodide/Emscripten编译的CPython
- **标准库支持**：builtins、math、json等核心模块
- **代码执行**：支持完整脚本和代码片段执行
- **输出捕获**：捕获stdout/stderr并显示在控制台

#### 2.1.3 调试功能
- **断点设置**：点击行号设置/取消断点
- **单步执行**：step into/step over/step out
- **继续执行**：从断点处继续运行
- **暂停执行**：强制暂停正在运行的代码
- **重启调试**：重新开始调试会话

#### 2.1.4 变量监视器
- **局部变量**：实时显示当前作用域的局部变量
- **全局变量**：显示全局命名空间中的变量
- **变量展开**：支持复杂对象（字典、列表、类实例）的层级展开
- **值变化高亮**：变量值变化时高亮显示
- **类型信息**：显示变量的类型信息

#### 2.1.5 调用栈可视化
- **调用栈显示**：以树形结构显示当前调用栈
- **帧信息**：显示每个栈帧的函数名、行号、文件名
- **帧切换**：点击栈帧切换到对应的代码位置
- **高亮当前帧**：当前执行的栈帧高亮显示

#### 2.1.6 虚拟文件系统
- **WASI集成**：通过WASI接口访问虚拟文件系统
- **文件管理**：创建、读取、写入、删除文件
- **目录操作**：创建目录、列出目录内容
- **文件持久化**：基于浏览器存储的文件持久化

#### 2.1.7 包管理
- **预编译包**：提供numpy等常用包的WASM预编译版本
- **包加载**：动态加载预编译的Python包
- **包信息**：显示已加载包的版本和信息

### 2.2 用户界面

#### 2.2.1 整体布局
- **左侧面板**：代码编辑器（占主要空间）
- **右侧面板**：调试工具区（可折叠）
  - 变量监视器
  - 调用栈
  - 断点列表
- **底部面板**：控制台输出（可折叠）
- **顶部工具栏**：运行/调试控制按钮

#### 2.2.2 交互设计
- **断点指示**：行号旁的红点表示断点
- **执行位置**：黄色箭头指示当前执行行
- **悬停提示**：鼠标悬停变量显示当前值
- **键盘快捷键**：支持常用调试快捷键（F5、F10、F11等）

## 3. 非功能需求

### 3.1 性能要求
- 编辑器加载时间 < 2秒
- Python运行时初始化 < 3秒
- 调试操作响应时间 < 100ms
- 支持至少1000行代码的编辑和调试

### 3.2 兼容性要求
- 支持Chrome、Firefox、Safari、Edge最新版本
- 支持WebAssembly和WASI的现代浏览器
- 响应式设计，支持不同屏幕尺寸

### 3.3 安全性要求
- 代码在沙箱环境中执行
- 无法访问本地文件系统（除虚拟文件系统外）
- 无法执行网络请求（除非显式授权）

### 3.4 可用性要求
- 直观的用户界面，降低学习曲线
- 清晰的错误提示信息
- 完整的功能文档和使用指南

---

## 4. 技术约束

### 4.1 前端技术栈
- React 18 + TypeScript
- Vite 构建工具
- Monaco Editor 代码编辑器
- Pyodide WASM 运行时
- TailwindCSS 样式框架

### 4.2 后端技术栈
- Node.js + Express
- 提供静态文件服务
- 预编译包管理和分发

### 4.3 关键技术
- WebAssembly (WASM)
- WebAssembly System Interface (WASI)
- Python调试协议（基于bdb）
- IndexedDB 本地存储
