## 1. 产品概述

股票技术分析工具是一个用于股票数据分析和可视化的 Web 应用，帮助用户通过 K 线图和布林带指标进行股票技术分析。

- 主要用途：提供股票历史 OHLCV 数据的可视化展示，支持布林带技术指标分析
- 目标用户：股票投资者、技术分析师、金融学习者
- 产品价值：将复杂的股票技术分析通过直观的图表呈现，帮助用户做出更明智的投资决策

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 浏览股票 K 线图、查看布林带指标、使用缩放平移功能 |

### 2.2 功能模块
1. **主页面**：股票选择、K 线图表展示、布林带指标叠加、数据控制面板
2. **指标参数配置**：布林带周期、标准差倍数调整

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主页面 | 股票选择器 | 选择不同股票代码进行分析 |
| 主页面 | K 线图表 | 展示股票历史 OHLCV 数据的蜡烛图 |
| 主页面 | 布林带曲线 | 在 K 线图上叠加显示上轨、中轨、下轨三条曲线 |
| 主页面 | 成交量柱状图 | 在图表下方展示对应时期的成交量 |
| 主页面 | 交互控制 | 支持鼠标滚轮缩放、拖拽平移图表 |
| 主页面 | 参数设置 | 调整布林带计算参数（周期、标准差倍数） |

## 3. 核心流程

用户打开应用 → 系统自动加载模拟股票数据 → 后端计算布林带指标 → 前端渲染 K 线图和布林带 → 用户可缩放/平移查看不同时间段 → 用户可调整指标参数重新计算

```mermaid
flowchart LR
    A["用户打开页面"] --> B["前端请求股票数据"]
    B --> C["Python 后端生成/获取 OHLCV 数据"]
    C --> D["Pandas 计算布林带指标"]
    D --> E["返回 JSON 格式数据"]
    E --> F["ECharts 渲染 K 线图和布林带"]
    F --> G["用户交互（缩放/平移/调参）"]
    G --> H["实时更新图表显示"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深色主题，深蓝灰色背景 (#0f1419)，突出金融数据的专业感
- **辅助色**：绿色 (#16c784) 表示上涨，红色 (#ea3943) 表示下跌，蓝色 (#3b82f6) 表示布林带
- **按钮样式**：圆角矩形，悬停有微动画效果
- **字体**：使用 JetBrains Mono 等宽字体展示数据，Inter 作为界面字体
- **布局风格**：顶部导航栏 + 主图表区域 + 侧边参数面板
- **图标风格**：简洁线性图标，避免过度装饰

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|----------|
| 主页面 | 顶部导航 | Logo、股票代码选择器、时间范围选择 |
| 主页面 | 图表区域 | K 线蜡烛图、布林带三线、成交量柱状图、时间轴 |
| 主页面 | 参数面板 | 布林带周期滑块、标准差倍数输入、重新计算按钮 |
| 主页面 | 数据图例 | 各指标颜色说明、当前价格显示 |

### 4.3 响应式设计
- 采用桌面优先设计，适配 1280px 及以上屏幕
- 移动端采用垂直堆叠布局，参数面板可折叠
- 触摸设备支持双指缩放和单指平移操作

### 4.4 交互细节
- 鼠标悬停 K 线显示详细数据 tooltip（日期、开高低收、成交量）
- 图表缩放时有平滑过渡动画
- 参数调整后图表实时重新计算和渲染
